Tic Tac Toe 

A simple and interactive Tic Tac Toe game built with React. Play against a friend on the same device and enjoy a clean, responsive user interface.

Features

* Two-player gameplay (X vs O)
* Winner detection
* Draw detection
* Game restart functionality
* Responsive design
* Built with React and modern JavaScript

Technologies Used

* React
* JavaScript (ES6+)
* CSS3
* Vite
